<h1>MiniTikTok</h1>
这是zhc和cyc的字节跳动大作业！<br>

<h3>上手指南</h3>
以下指南将帮助你在本地机器上安装和运行该项目，进行开发和测试。<br>

<h4>安装要求</h4>
1.安装andriod studio

<h4>安装步骤</h4>
1.clone到本地<br>
2.用andriod studio打开<br>
3.运行<br>

<h4>程序特点</h4>
1.使用viewPager实现主页面的UI设计<br>
2.使用animator实现recyclerView的scroll动画<br>
3.视频录制按钮随手指移动，并可实时调焦距<br>
4.自动生成视频缩略图<br>
5.个人主页查看已经发送的视频<br>
